
1.application-common.xml 
    Spring一些公共配置，如dataSource，事物处理等
    
 2.application-controller.xml（可删除）
   配置Spring MVC的controller，类似struts的Action配置文件，不过已经设置不用配置controller，文件仅供参考，不需要可以删除
   
3.application-service.xml（可删除）
  配置service，已经设置不用配置service，文件仅供参考，不需要可以删除
  
 4.application-task.xml（可删除）
   配置定时任务，这个比较有用，大家可以看看，比Spring2.x的版本简单了很多 ，文件仅供参考，不需要可以删除
   
   
4.spring-servlet.xml
   这比较重要Spring MVC的配置，大家可以看看相关资料。配置方法。
